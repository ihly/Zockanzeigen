1. Das Git-Inventory muss in C:\xampp\htdocs geklont werden (git clone git@github.com:ihly/Zockanzeigen.git)

2. Über das XAMPP Control Panel muss Apache und MySQL gestartet werden.

3. phpMyAdmin muss geöffnet werden und im Tab "Importieren" die Datenbank "image_upload.sql" importiert werden.

4. Über index.html kann die Webanwendung gestartet werden.
